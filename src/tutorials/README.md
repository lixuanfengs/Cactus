# 实战教程

这里收录了各种实战项目和工具使用教程，通过实际案例帮助你掌握技术应用。

## 📚 教程分类

### 🚀 项目实战
- **AI-MCP-CactusLi** - 基于Spring AI MCP的智能自动发帖系统
- **Spring AI RAG** - AI知识库项目完整实现
- **OAuth2.0** - 认证授权系统实战
- **性能优化** - Java项目性能调优实践
- **延时任务** - 多种延时任务实现方案

### 🛠️ 工具使用
- **Maven构建** - 项目构建和依赖管理
- **网络工具** - 各种网络工具的使用技巧
- **调试工具** - Fiddler等抓包工具使用

## 🎯 学习特色

- **完整项目** - 从0到1的完整实现过程
- **实用工具** - 提高开发效率的工具技巧
- **最佳实践** - 行业标准的实现方案
- **问题解决** - 常见问题的解决方案

## 📖 推荐学习方式

1. **选择感兴趣的项目** - 根据技术栈选择合适的教程
2. **跟随实践** - 边学边做，动手实现
3. **理解原理** - 不仅要知道怎么做，还要知道为什么
4. **扩展应用** - 在理解基础上进行创新和扩展
